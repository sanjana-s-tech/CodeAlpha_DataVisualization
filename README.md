# CodeAlpha Data Visualization

## Objective
Transform raw financial data into meaningful visualizations and insights.

## Dataset
Financial Sample.xlsx

## Tools Used
- Python
- Pandas
- Matplotlib
- OpenPyXL

## Visualizations Created
1. Total Sales by Country
2. Profit by Product
3. Sales Distribution by Segment
4. Monthly Sales Trend
5. Profit by Country
6. Product Sales Comparison

## Key Insights
- United States generated the highest sales.
- Mexico generated the lowest sales.
- Product profitability varies significantly.
- Sales trends differ across months.

## How to Run

pip install pandas matplotlib openpyxl

python visualization.py
